# OpenCalc

### A simple and beautiful calculator for Android.

## 📖 Features

* Lightweight (APK ~2MB)
* No unnecessary permissions
* Dark, Light, Amoled and Material You theme
* Scientific mode
* History
* Portrait and landscape orientation
* Quick settings tile
* Available on locked screen
* More...

## ⚠️ Requirements

* Android 5.0+
